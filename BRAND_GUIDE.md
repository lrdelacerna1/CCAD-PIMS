# CCAD PIMS Brandkit

This guide outlines the visual identity for the CCAD Personal Information Management System (PIMS), derived from the official brandbook.

---

## 1. Typography

### Primary Font Stack
**Font Family:** Helvetica, Verdana, Arial, sans-serif  
**Usage:** All text elements (Body, Headings, Menus, UI).

### Text Styles

**Body Text (<p>)**
*   **Color:** `#313131` (Dark Grey)
*   **Size:** `12px`
*   **Line Height:** `25px`
*   **Weight:** `400` (Regular)
*   **Alignment:** Justify
*   **Smoothing:** Antialiased

**Heading 2 (<H2>)**
*   **Color:** `#000000` (Black)
*   **Size:** `20px` (Mobile) / `25px` (Tablet) / `30px` (Desktop)
*   **Line Height:** `26px`
*   **Weight:** `700` (Bold)

## 3. UI Components

### Navigation / Menu Tabs
*   **Font Family:** Helvetica, Verdana, Arial, sans-serif
*   **Font Size:** `13.68px`
*   **Weight:** `700` (Bold)
*   **Color:** `#000000`
*   **Hover Color:** `#880000` (Red)
*   **Padding:** `0px` (top/bottom), `20px` (left/right)
*   **Cursor:** Pointer

### Buttons
*   **Primary Background:** `rgb(182, 183, 181)` (Gray)
*   **Primary Text:** `#ffffff` (White)
*   **Secondary Background:** `#ffffff` (White)
*   **Secondary Text:** `#000000` (Black)
*   **Danger Text:** `#880000` (Red) - Background is white
*   **Font:** Helvetica, Verdana, Arial, sans-serif
*   **Size:** 12px
*   **Weight:** 600 (Bold)
*   **Text Transform:** Uppercase
*   **Height:** 30px
*   **Border Radius:** 12px
